# Earth & Times — Design System

## Visual principle
Ambient instrument, not interface.

## Palette
- Void: #0B0F12
- Abyss: #10242B
- Dusk: #274C57
- Haze: #6C8D8D
- Ash: #B7B2A3
- Bone: #E8E0CF
- Ember: #E88E68
- Apricot: #F1B085
- Rose Dust: #CC7D72
- Moonwash: #C8D1D4

## Typography
- Cormorant Garamond for headings and display
- IBM Plex Mono for metadata

## Do
- keep layouts sparse
- preserve negative space
- use calm editorial structure
- keep metadata restrained

## Don’t
- build like a dashboard
- overuse cards
- over-label the map
- use neon sci-fi effects
