# Earth & Times — Project Spec

## Product Summary
Earth & Times is a living biblical clock and witnessing tool.

It presents a fullscreen, immersive observatory-style interface that renders biblical time, scriptural cosmology, and historical time as a contemplative visual experience.

The app is not a generic research library and not a dashboard.

## One-line positioning
A living biblical clock with layered pathways into scriptural time, scriptural cosmology, and scriptural history.

## Experience hierarchy
1. beauty
2. relevance
3. explanation
4. argument
